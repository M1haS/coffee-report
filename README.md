# Coffee Report

Генерация отчётов по данным о подготовке студентов к экзаменам из CSV-файлов.

## Описание

Скрипт обрабатывает CSV-файлы с данными о студентах (траты на кофе, сон, часы учёбы и т.д.) и формирует отчёты. 

Текущий доступный отчёт: **median-coffee** — медианная сумма трат на кофе по каждому студенту за весь период.

## Установка

```bash
uv sync --extra dev
```

## Запуск

```bash
# Базовый пример
uv run coffee-report --files data.csv

# С указанием типа отчёта
uv run coffee-report --files data.csv --report median-coffee

# Несколько файлов
uv run coffee-report --files file1.csv file2.csv file3.csv --report median-coffee
```

## Формат входных данных

CSV-файл должен содержать колонки:
- `student` — имя студента
- `date` — дата
- `coffee_spent` — сумма трат на кофе
- и другие колонки (опционально)

Пример:
```csv
student,date,coffee_spent,sleep_hours,study_hours,mood,exam
Алексей Смирнов,2024-06-01,450,4.5,12,норм,Математика
Дарья Петрова,2024-06-01,200,7.0,6,отл,Математика
```

## Пример вывода

```
+---------------------+---------------------+
| Student             |   Median Coffee Spent |
+=====================+=====================+
| Иван Кузнецов       |              650.00 |
+---------------------+---------------------+
| Алексей Смирнов     |              500.00 |
+---------------------+---------------------+
| Дарья Петрова       |              250.00 |
+---------------------+---------------------+
```

## Архитектура

- `data_loader.py` — загрузка и парсинг CSV-файлов
- `reports.py` — генерация отчётов (расширяемая архитектура через Registry)
- `cli.py` — CLI интерфейс на argparse

Для добавления нового отчёта:
1. Создайте функцию-генератор отчёта в `reports.py`
2. Зарегистрируйте её через `registry.register("report-name", function)`
